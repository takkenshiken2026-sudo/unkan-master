包装貨物の荷扱い指示マーク（JIS Z 0150 ：2001）と指示内容について、次のうち誤っているものはどれか。( 指示マーク ) ( 指示内容 )

問題

包装貨物の荷扱い指示マーク（JIS Z 0150 ：2001）と指示内容について、次のうち誤っているものはどれか。( 指示マーク ) ( 指示内容 )

選択肢

正答

正答は （4） です。

解説

正解の理由

荷扱い指示マークは、マークの形と指示内容を対応させて判断する。選択肢４のマークは「直射日光にさらしてはならない」という指示内容とは対応していないため誤りである。壊れもの、積重ね段数、手かぎ禁止の対応は適切である。

（4） 包装貨物を直射日光にさらしてはならない。

他の選択肢

  • （1） 包装貨物の中身は壊れやすい。そのため注意して取り扱わねばならない。

    「包装貨物の中身は壊れやすい。そのため注意して取り扱わねばならない。」は、一見もっともらしく見える場合がありますが、正答（4）「包装貨物を直射日光にさらしてはならない。」と比べると、学習・制度・実務の観点で「最も問題がある」記述ではありません。

    「最も適切でない」形式では、正しそうな肢が複数あることがあります。各肢の主語・客体・数字・期限・手続の順序が設問条件と合うかを確認し、最も不適切な一つだけを選びます。

  • （2） 同一包装貨物を上に積み重ねる場合の最大積重ね段数を示す。ｎは制限する段数である。n

    「同一包装貨物を上に積み重ねる場合の最大積重ね段数を示す。ｎは制限する段数である。n」は、一見もっともらしく見える場合がありますが、正答（4）「包装貨物を直射日光にさらしてはならない。」と比べると、学習・制度・実務の観点で「最も問題がある」記述ではありません。

  • （3） 包装貨物を取り扱う際、手かぎの使用を禁止する。

    「包装貨物を取り扱う際、手かぎの使用を禁止する。」は、一見もっともらしく見える場合がありますが、正答（4）「包装貨物を直射日光にさらしてはならない。」と比べると、学習・制度・実務の観点で「最も問題がある」記述ではありません。

学習のヒント

この問題で間違えた場合は、設問文の求め方（「正しいもの」「誤っているもの」「最も適切でないもの」）を最初に線引きしてください。正答・誤答それぞれについて、用語の定義と制度の前提を用語解説で確認し、復習リストや実践演習・一問一答と組み合わせて、同分野の過去問を解き直すと定着しやすくなります。
