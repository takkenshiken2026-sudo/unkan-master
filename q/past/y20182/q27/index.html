問題

自動車の走行時に生じる諸現象とその主な対策に関する次の文中、Ａ、Ｂ、Ｃ、Ｄに入るべき字句としていずれか正しいものを１つ選びなさい。ア （Ａ）とは、路面が水でおおわれているときに高速で走行するとタイヤの排水作用が悪くなり、水上を滑走する状態になって操縦不能になることをいう。これを防ぐため、日頃よりスピードを抑えた走行に努めるべきことや、タイヤの空気圧及び溝の深さが適当であることを日常点検で確認することの重要性を、運転者に対し指導する必要がある。

選択肢

  1. （1） 制動距離
  2. （2） 空走距離

正答

正答は （A-1;B-2;C-1;D-2） です。

解説

正答の組合せ：A→（1）制動距離、B→（2）空走距離、C→（1）制動距離、D→（2）空走距離

正解の組合せ

  • A → （1） 制動距離

  • B → （2） 空走距離

  • C → （1） 制動距離

  • D → （2） 空走距離

Ａ．路面が水でおおわれ、高速走行時に水上を滑走する状態になる現象はハイドロプレーニング現象である。Ｂ．夜間、自車と対向車のライトにより歩行者等が見えにくくなる現象は蒸発現象である。Ｃ．ブレーキ液内に気泡が発生して制動力が低下する現象はベーパー・ロック現象である。Ｄ．危険を認知してからブレーキ操作に至るまでに走行する距離は空走距離である。
