運行管理者試験 実践演習 第306問（労働基準法関係）

問題

改善基準の拘束時間について、次のＡ、Ｂ、Ｃ、Ｄに入るべき字句を選択肢から選びなさい。拘束時間は１年（Ａ）以内かつ１ヵ月（Ｂ）以内が原則である。労使協定がある場合でも、１年（Ｃ）以内の範囲で１ヵ月（Ｄ）までの延長に限られる。

選択肢

  1. （1） 3,300時間
  2. （2） 3,400時間
  3. （3） 284時間
  4. （4） 310時間
  5. （5） 293時間
  6. （6） 320時間
  7. （7） 3,516時間
  8. （8） ６ヵ月

正答

正答は （A-1;B-3;C-2;D-4） です。

解説

正解の組合せ

  • A → （1） 3,300時間

  • B → （3） 284時間

  • C → （2） 3,400時間

  • D → （4） 310時間

現行の改善基準では、拘束時間は１年3,300時間・１ヵ月284時間が原則で、労使協定による例外でも１年3,400時間・１ヵ月310時間が上限である。
