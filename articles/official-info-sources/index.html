運行管理者試験の公式情報の確認先｜要項・日程・合格基準

運行管理者試験の数値・日程・合格基準は、公益財団法人 運行管理者試験センター（https://www.unkan.or.jp/）が公表する受験要項と試験案内が唯一の正本です。ブログやSNSの「最新情報」は、必ず公式ページと照合してから採用してください。例えば受験手数料6,000円やシステム利用料660円は年度で改定されることがあり、申込の1週間前に要項PDFを開き直す習慣が安全です。残り12週で学習を始める場合も、最初の30分は公式確認に充てると後から計画を組み直す手戻りが減ります。

この記事の要点

本記事を読むと、どの公式ページをどの順序で確認すればよいかが分かり、非公式情報に惑わされずに申込・学習・CBT予約まで進められます。

1一次情報の確認先一覧

学習中も申込前も、参照順は次のとおりが基本です。

情報正本主な確認内容
受験要項試験センター公式資格・科目・合格基準・手数料
試験日程同上申込期間・CBT実施期間
公式テキスト試験センター発行5科目の章立て・改訂版
法令e-Gov等条文の最新版（要項の引用元）

例えば「各科目60％以上」という合格基準は、要項の「合格の判定」章に表形式で載っています。PDFの該当ページを印刷し、学習机の横に置くと迷いにくくなります。

2要項で必ず写す項目

要項から次の項目をメモ帳に転記してください。

  • 受験資格（実務経験・選任要件）
  • 受験区分（貨物 / 旅客）
  • 5科目名と1科目あたりの問題数
  • 科目ごとの合格基準（正答率・得点）
  • 受験手数料・システム利用料・支払方法
  • 申込期間・CBT予約の流れ

例えば1科目20問で合格基準60％なら、12問以上正解で合格ラインに乗ります。演習では科目ごとに「12/20」を週次で記録すると弱点が見えます。

3CBT・会場予約の確認ポイント

CBT受験では、申込後に会場・日時の予約が必要な場合があります。

確認項目理由
予約サイトのURL非公式リンクのフィッシング回避
変更・取消期限試験日直前のトラブル防止
持ち物・禁止物当日の受験不可リスク低減
操作説明本番前の慣れ（模擬操作があれば利用）

例えば試験2週間前に、会場までの交通と開始30分前到着の出発時刻をカレンダーに入れておくと、当日の焦りを減らせます。

4非公式情報との照合手順

検索結果や動画の数値は、次の手順で採用可否を決めます。

1. 要項PDFの該当箇所を開く
2. 科目名・手数料・合格率の数値を突合
3. 不一致なら公式を正として学習計画を更新
4. 古い年度の要項を引用していないか発行年を確認

たとえば「受験料5,000円」と書かれた古い記事は、現行要項の6,000円と異なるため、そのまま予算計画に使わないでください。

5更新時に見直すタイミング

次のタイミングで公式ページを再確認します。

  • 学習開始時（最初の1時間）
  • 申込開始日
  • 申込完了直後（CBT予約）
  • 試験1か月前（要項改定の有無）
  • 試験1週間前（会場・持ち物）

例えば毎月1日に要項ページを開き、受験手数料6,000円や合格基準60％の記載が前回と同じかだけ確認する習慣でも、直前の情報差し替えに気づきやすくなります。試験1週間前は会場住所も再確認してください。

6よくある質問

公式情報はどこで確認しますか？
公益財団法人 運行管理者試験センター（https://www.unkan.or.jp/）が正本です。受験要項・試験日程・CBT案内・合格発表方法はすべて公式サイトで確認してください。ブログのまとめは参考程度に留め、数値は必ず要項PDFと照合します。
5科目の合格基準はどこに書いてありますか？
受験要項の「試験方法」「合格の判定」などの章に、科目ごとの問題数と合格基準が表形式で載っています。例えば各科目60％以上など、年度で文言が変わるため、当年度のPDFをダウンロードして該当表を直接開いてください。
公式テキストの版はどう確認しますか？
試験センターが指定する最新版の表示（発行年・版数）を要項またはテキスト案内ページで確認します。古い版のまま学習すると、改正法令の論点が試験範囲とずれることがあるため、購入前と試験2か月前の2回、版表示を照合することをおすすめします。
