運行管理者試験｜初めて受験する人向けガイド

初めて運行管理者試験を受ける場合、「要項確認→区分決定→テキスト1周→科目別演習→CBT予約」の順序を固定すると迷いが減ります。CBTで5科目×各20問、科目ごとに60％以上が一般的な合格基準（要項で再確認）です。例えば試験日まで残り16週あるなら、最初の4週でテキスト通読、次の8週で演習中心、最後4週で弱点科目の解き直しという3段階が組みやすいです。受験手数料6,000円とシステム利用料660円も、申込時期と一緒にカレンダーへ入れてください。

この記事の要点

本記事を読むと、初受験者が最初の1か月で何をすべきかが分かり、5科目すべてを合格ラインに乗せるための初期計画を立てられます。

1初受験前の4ステップ

初受験は次の4ステップを順番に進めます。

1. 要項で資格・区分・合格基準を確認
2. 公式テキスト＋演習1系統を決める
3. 5科目を1周する（完璧を目指さない）
4. 科目別正答率で弱点を特定し、再演習

例えば1週目は事業法のみテキスト第1章と演習10問、2週目は車両法…と科目ローテにすると、5週で1周できます。残り期間は弱点科目に時間を寄せます。要項の合格基準60％を各科目の演習記録に必ず書き添えてください。

2最初の2週間でやること

最初の2週間の目標は「5科目の全体像」と「自分の弱点1科目」の発見です。

日目安内容
1〜2日目2時間要項精読・メモ
3〜7日目平日30分×5テキスト目次と用語10語
8〜14日目週末3時間×2科目別演習20問×2科目

たとえば道交法だけ正答率50％、他科目70％なら、3週目以降は道交法に週＋2時間を配分します。初月から1科目を極端に放置しないことが重要です。

3科目別演習の始め方

演習は科目タグ付きで解き、ノート1行形式で記録します。

記録例：「第8問／労基法／36協定の上限日数を混同」

例えば20問中12問正解なら60％で合格ライン上、11問なら59％で不合格ライン下です。1問差を意識して解き直すと、本番の足切りリスクを体感できます。運管マスターの演習・一問一答は科目フィルタで回せます。週1回は5科目すべての正答数を1行にまとめると、放置科目に早く気づけます。

4CBT・申込のタイミング

申込とCBT予約は別タイミングのことがあります。

  • 申込期間内にInternet申込・手数料支払
  • 案内に従いCBT会場・日時を予約
  • 受験票・確認メールを保存

たとえば申込締切の3日前ではなく、開始1週間以内に6,000円の手数料支払まで済ませると、CBT予約の選択肢が残りやすくなります。予約後は試験日から逆算して演習量を固定してください。申込完了メールは試験日まで保管しておくと安心です。

5初受験者が陥りやすいミス

初受験者がよく失敗するのは次のパターンです。

- 1. 区分（貨物/旅客）を間違えて教材を買う
2. 実務だけ先にやり
- 労基法の数値が足りない
3. 科目合格制を忘れ
- 総得点だけを気にする
4. CBT予約を忘れ
- 申込だけで終わる

例えば4科目で70％超えでも、車両法が55％なら不合格です。週次で5科目すべての正答数を1行記録する習慣を早めに始めてください。

6よくある質問

初心者は何から手を付ければよいですか？
運行管理者試験センターの要項で資格と区分を確認したうえ、公式テキストの目次を5科目分写し、最も身近な科目（多くは実務または事業法）から1章＋演習10問を始めるのがおすすめです。最初の1週間で5科目すべてに触れると、後の計画が立てやすくなります。
初受験でCBTは不安です。対策は？
科目ごと20問の演習をPCまたはタブレットで行い、1問あたり3分以内を目安に解く練習をしてください。例えば5科目通して100問・180分の想定で月1回タイム計測すると、本番の時間配分に慣れます。会場の操作説明は予約確認ページを事前に読んでください。
独学で初受験は可能ですか？
可能です。公式テキスト＋演習＋用語解説の3点セットで進め、科目ごとの正答率を週次記録してください。例えば週10時間なら、弱点科目に4時間・その他4科目に各1.5時間など、60％未満の科目から時間を寄せると合格ラインに届きやすくなります。
