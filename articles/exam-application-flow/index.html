運行管理者試験の申込みの流れ｜Internet申込からCBT予約まで

運行管理者試験の申込は、運行管理者試験センターの申込サイトが正本です。おおむね「アカウント作成→受験区分（貨物/旅客）選択→必要情報入力→写真アップロード→6,000円＋システム利用料660円の支払→CBT会場・日時予約」の順で進みます。例えば申込期間の開始3日目に写真データと区分を確定しておくと、締切前の混雑を避けやすくなります。残り2週間で申込する場合も、最初の30分は要項の手順ページを開いたまま進めてください。

この記事の要点

本記事を読むと、申込からCBT予約までの全ステップが分かり、例えば6,660円支払完了後24時間以内に予約サイトへ進む流れを漏れなく実行できます。

  • 受験資格の確認
  • 年間スケジュール
  • 申込期間と手数料
  • 申込手順と会場

1申込前に揃えるもの

申込前に次を準備すると手戻りが減ります。

  • 氏名・生年月日（公的身分証と一致）
  • 受験区分：貨物 / 旅客
  • 写真データ（要項のピクセル・背景色）
  • 支払手段（コンビニ / クレジット等）
  • メールアドレス（確認メール受信可）

例えば写真背景が要項と1色違うだけで差し戻されることがあるため、申込開始前に規格サンプルと自分の画像を並べて確認してください。

2Internet申込の基本ステップ

申込サイトではおおむね次の順序で進みます。

ステップ内容
1利用規約・要項確認
2受験者情報入力
3区分・連絡先
4写真アップロード
5手数料6,000円＋利用料660円の決済
6CBT予約（案内どおり別画面の場合あり）

たとえばステップ5まで30分で終え、ステップ6は支払反映後に行う流れなら、同日中に会場・日時まで確定できることが多いです。

3写真・区分でつまずきやすい点

申込不備の多くは写真と区分です。

  • 写真：サイズ超過・背景色・顔の比率
  • 区分：貨物事業者なのに旅客を選択（テキスト・出題範囲がずれる）
  • 氏名：旧字体・ミドルネーム表記の不一致

例えば一般貨物自動車運送事業者の社員なら貨物区分が原則で、申込確認画面で「貨物」と表示されているか最終チェックしてください。区分ミスは申込後に直せない場合があります。旅客区分を選ぶと出題範囲が旅客向けテキストに切り替わるため、5科目の演習も貨物用に積んだ時間が無駄になりやすい点に注意してください。

4支払完了後のCBT予約

支払完了後はCBT予約へ進みます。

  • 予約サイトURLは申込完了メール・案内ページが正本
  • 会場は都道府県単位で選択→空き日時を選ぶ
  • 変更・取消期限を要項で確認

例えば土曜午前10時の枠が人気なら、支払完了翌日の朝9時に予約すると希望日を取りやすい傾向があります。予約番号と会場住所はスマートフォンのメモ帳にも写してください。予約後24時間以内に会場までの交通を調べ、試験開始90分前到着を目安に出発時刻をメモしておくと当日がスムーズです。

5申込後にやること

申込・予約完了後は学習モードに切り替えます。

1. CBT日から逆算した週次演習表を作成
2. 5科目の正答数を週1回記録
3. 会場までの交通ルートを試験2週間前に確認
4. 持ち物リストを要項から写す

たとえばCBTまで残り9週なら、週1回5科目×20問に近い演習を入れ、60％ライン（12問以上）に届いていない科目から時間を寄せてください。申込完了日を学習ログの起点日として記録すると、後から振り返ったときに週次ペースが把握しやすくなります。

6よくある質問

申込みは郵送でもできますか？
申込方法は当年度要項が正本です。例えばインターネット申込のみと案内されている2026年度要項では、郵送申込は受け付けられません。要項の「申込方法」章でWeb限定かどうかを必ず確認し、紙の申込書が必要な年度かどうかもPDF内検索で調べてください。
申込後に区分を変更できますか？
変更可否は要項・申込サイトの規定どおりです。例えば貨物と旅客の区分を誤ると出題範囲・テキストがずれるため、申込確定前の確認画面で3回チェックすることをおすすめします。変更不可の場合は取消・再申込が必要になることがあります。
CBT予約まで終われば申込完了ですか？
要項・案内では、手数料支払とCBT予約の両方が完了して初めて受験確定となることが多いです。例えば支払だけ済んで予約が未完了のまま期限を過ぎると受験できないケースがあるため、予約確認画面を保存し、1週間後にもう一度ログインして表示を確認してください。
