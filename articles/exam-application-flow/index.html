運行管理者試験の申込手順｜Internet申請から入金まで

新規受験の申込はインターネットだけで、書面の郵送はできません。申請後は7～10日の書類審査があり、その後CBTサイトで会場予約と6,660円の支払をします。審査が終わるまでCBTの予約はできない点に注意してください。

13つの入口 — 要項PDF・申請サイト・CBT専用

手続きは3つのWeb入口に分かれています。役割を混同すると、審査前に会場を予約しようとして止まります。

入口用途
新規受験申請マニュアル要項・受験案内PDF
インターネット受験申請新規申請・書類アップロード
CBT試験専用会場予約・6,660円支払

申込を送信した時点では、CBTサイトで会場は選べません。審査完了の案内メールが届くまで、マイページの審査状況だけを毎日確認します。

2STEP1申請 — 締切までに送る内容

令和8年第1回の申請受付期間はPDFを参照してください。締切週に慌てないよう、次を先に揃えます。

1. 本人のメールアドレス登録・確認
2. 受験区分（貨物/旅客）と資格ルートの入力
3. 本人確認書類・6か月以内の顔写真（JPEG・10MB以内）
4. 実務ルートなら承認者の5項目、講習ルートなら修了証など

たとえば締切の2週前に送信すると、「受験申請受付のお知らせ」メールが届きます。書類・承認の締切と、会場予約の締切は別の日付です（当回のPDFが正本）。

3審査7～10日 — 触るのはマイページだけ

新規受験の書類審査は1週間～10日ほどかかります。この間はCBTサイトを開けません。

日数できること
1～3日目承認者へリマインド（実務ルート）
4～7日目要項の30問配分表を暗記
8～10日目会場候補3つと支払口座の確認

例えば審査が終わる7～10日後に、「CBT試験会場予約等の手続きのご案内」メールが届く想定です。案内メールが来ない日は、マイページの審査状況を確認します。

4配付番号でログイン — 予約・6,660円・確認書メール

審査が終わると、マイページからCBT試験専用サイトへ進めます。ログインには次の3点が必要です（要項⑥）。

・申請者の配付番号（受付メールに記載）
・氏名（カナ）
・生年月日

審査完了の翌日に東京会場・10:00枠を取り、同日中に6,660円をクレジット決済します。入金が終わると受験確認書メールが届き、郵送はありません。支払後は返金できないため、日時の選択は慎重に行います。

5入金完了からCBT当日 — 30問90分の準備

受験確認書メールはスマホ表示でも持参できます。顔写真付きの本人確認書類も当日は必須です。

タイミング確認内容
予約直後会場名・日時・受付締切
CBT4週前90分・30問の模擬を1回
前日免許証・確認書メールの表示

CBT2週前には90分で30問を演習し、18/30以上を目標にします。申込完了は学習本番のスタート合図です。分野別8・4・5・6・7の列を付けた演習記録を、週1回更新してください。

6よくある質問

申込後すぐCBT会場を予約できますか？
いいえ。書類審査が終わって案内メールが届いてから予約できます。新規は7～10日、再受験は数営業日が目安です。申込直後はCBTサイトを開けないのが普通の状態です。マイページの審査状況を毎日確認してください。
書面申込はできますか？
できません。インターネット申請のみです。申込開始日にPCまたはスマホから受験申請サイトへアクセスします。書面郵送の案内は現行の要項にないため、トップページの申込方法を当年度のPDFで確認してください。
同じ回で貨物と旅客の両方を申込めますか？
いいえ。1回の募集で申し込めるのは、貨物か旅客のどちらか一方です。令和8年第1回で貨物30問を受けたら、同じ回で旅客30問は申込めません。両方が必要なら、別の回で受験する計画を立ててください（要項で再確認）。
