運行管理者試験の車両法基礎｜点検・整備・4問

車両法基礎は第2分野4問向けです。要項④の道路運送車両法関係から、登録・検査・点検・整備・記録が中心の論点です。出題は4問と最も少ない一方、1問ミスで②未達に直結しやすい分野です。30問CBTでは、（2）4問中1問以上が合格②の条件です。貨物・旅客共通なので、本記事1本に固定してください。

1最少分野の（2）4問 — 0/4が②未達に直結

第2分野は全30問中4問で、出題比は約13%です。②は4問中1問以上の正解が必要で、0/4は即②未達です。範囲は道路運送車両法の登録・検査・点検・整備・記録に集中します。

指標目安
出題4問/30問
②1/4以上
安定3/4以上
週演習6～8問

車両法の目的は所有権の公証・安全性の確保・公害の防止です。合計が19/30でも（2）が0/4なら不合格圏です。数値・周期・区分の暗記が得点に直結する分野です。

2登録と検査 — 4種の登録・車検の年数を覚える

登録は4種に分かれます。

  • 新規は新車や一時抹消車の再使用
  • 移転は所有者の変更
  • 変更は使用者の住所・氏名や使用の本拠の変更
  • 抹消は使用の中止や解体・輸出

所有者の変更が移転、それ以外の変更が変更登録という切り分けが頻出のひっかけです。

検査の種類は新規・継続・臨時・構造等変更・予備の5つです。車検証の有効期間は用途・車種で異なります。

区分初回継続
事業用貨物8t未満2年1年
事業用貨物8t以上1年1年

事業用貨物車は継続車検が毎年です。8t未満の初回2年と取り違える肢に注意します。

3日常点検と定期点検 — 周期と実施者を区別

点検整備は日常点検と定期点検に分かれます。

事業用自動車で。

  • 運行する者が1日1回
  • 運行開始前に日常点検を行い

マイカーが走行距離等で適切な時期に行うのと違い、毎日が義務です。

定期点検は周期が固定されています。

種別周期項目数の目安
日常点検1日1回運行前運行に応じ確認
定期点検3か月ごと約50項目
定期点検12か月ごと約100項目

事業用車の点検整備記録簿の保存期間は1年です。3か月点検と12か月点検を入れ替える肢が頻出のひっかけです。

4整備管理者と保安基準 — 選任要件と装置要件

整備管理者は、車庫管理や点検整備を処理させるため一定の要件を満たす者から選任します。

要件。

  • 整備士技能検定の合格者
  • または点検整備の実務2年以上で選任前研修を受けた者など

業務。

  • 日常点検の実施方法を定めること
  • その結果から運行可否を決めること
  • 定期点検の実施など

地方運輸局長は、保安基準に適合しない恐れがあるとき整備命令を出せます。

保安基準は灯火・制動・速度抑制など装置の技術要件です。道路運送車両法の種別は普通・小型・軽・大型特殊・小型特殊の5つで、道路交通法の免許区分とは別体系です。両者の混同に注意します。

5週次演習 — （2）タグを6～8問維持

（2）分野は出題4問に対し、週6～8問が維持の下限です。数値暗記が点になるため、登録4種・車検年数・点検周期・整備管理者の4論点を回します。

1. 市販集の車両法章4問
2. past.html同分野2問
3. 解説→テキスト章20分
4. 類題2問

30問計測で（2）が0/4だった週は、翌週を15問に増やします。登録・検査・点検整備から各5問という配分が立て直しの定番です。過去問3ステップと分野別20問/週記事を併用してください。

630問計測 — （2）4問行の読み方

30問90分の計測では、（2）4問行を単独で評価します。

・3/4以上：安定域
・2/4：②充足・要強化
・1/4以下：週15問へ
・0/4：最優先20問/週

本番56日前までに、（2）3/4と②1回以上を達成しましょう。週末の8/4/5/6/7行で、2週連続の確認を取ります。（2）が2週横ばいなら、新規を止めて解き直し50%へ切り替えます。合格は18/30と分野②の両方が必要です。学習計画の詳細は学習計画記事へ委ねます。

7よくある質問

4問だけなのに時間をかけますか？
②リスクが高いため、時間をかけます。出題比13%でも、0/4は即②未達で18/30の合格には届きません。登録・車検年数・点検周期は暗記で確実に取れるので費用対効果が高いです。週6～8問を維持し、0/4の週は15問にするのが目安です。
事業用トラックの車検は何年ごとですか？
車両総重量8トン以上の事業用貨物車は、初回も継続も1年ごとです。8トン未満は初回2年・継続1年です。自家用乗用とは異なり、いずれも継続車検が毎年になる点が試験のひっかけです。検査の種類は新規・継続・臨時・構造等変更・予備の5つも合わせて覚えます。
日常点検と定期点検はどう違いますか？
日常点検は事業用車なら1日1回、運行開始前に行う簡易な確認です。定期点検は3か月ごと・12か月ごとに行うやや大がかりな点検整備です。点検整備記録簿は事業用で1年保存します。3か月と12か月の周期や、実施者・記録の組み合わせを入れ替える肢が頻出です。
