運行管理者試験の難易度｜合格率30〜40％と科目足切り

運行管理者試験の難易度は、公表合格率30〜40％前後と5科目各60％の科目合格制の両面で理解する必要があります。実務経験者は実務科目は得点しやすい一方、労基法・車両法の数値問題で11/20に留まりやすい傾向があります。例えば残り12週・週10時間なら、演習正答率が最低の2科目に各3時間、残り3科目に各1時間ずつ配分すると足切りリスクを下げやすくなります。

この記事の要点

本記事を読むと、難易度の正体が「科目合格制」にある理由が分かり、例えば実務15/20でも労基法10/20なら対策優先順位を入れ替えられます。

  • 合格率の確認先
  • 合格点・基準の意味
  • 難易度の考え方
  • 統計の読み方
  • 学習計画への反映

1難易度を決める2つの軸

難易度は次の2軸で考えます。

軸内容数値例
公表合格率全体の合格割合30〜40％前後
科目合格制5科目各60％12/20×5科目
個人演習自分の正答率週次記録

例えばR7第1回貨物37.2％（公表値）でも、自分の演習で5科目すべて13/20以上なら個人の合格見込みは公表値より高いと判断しやすくなります。

2科目別のつまずきポイント

初学者・経験者ともに難しやすい科目です。

科目難所対策
労基法時間・36協定の数値数値カード＋20問/週
車両法点検周期・検査表を1枚にまとめる
事業法許可・届出の主体条文＋演習
道交法装備・速度誤答ノート
実務手順の例外現場メモと照合

たとえば労基法20問で8/20（40％）なら、残り8週の半分を労基法に回し、12/20達成まで新規科目を増やさない方が効率的です。

3実務経験者の落とし穴

現場経験者が陥りやすいパターンです。

1. 実務科目だけ15/20以上で安心する
2. 法令の数値暗記を後回しにする
3. 「現場ではこうだから」で肢を選ぶ
4. 演習量不足（週5問以下）

例えば実務17/20・事業法14/20でも、労基法10/20・車両法9/20なら不合格です。残り10週なら法令2科目に週＋4時間を先に配分し、3月15日CBTまでに5科目すべて12/20以上を達成する計画に切り替えてください。

4時間配分で難易度を下げる

難易度は時間配分で調整できます。

残り期間配分例（週10時間）
12週弱点2科目×3h＋他3科目×1h＋復習1h
8週弱点2科目×4h＋100問通し2h
4週解き直し6h＋通し100問4h

例えば3月15日CBTまで8週なら、第1週に5科目20問ずつ解き、正答数が最低の2科目を特定してから上表どおり配分を固定します。

5難易度と合格率の誤解

難易度についての誤解です。

1. 合格率60％超の旧データが今も当てはまる
2. 実務経験があれば勉強不要
3. 総合正答率が高ければ合格
4. 旅客と貨物の難易度が同じ

たとえば2020年代の公表値30〜40％でも、5科目すべて14/20以上を8週続ければ十分に合格圏内に入れます。判断材料は公表値より、3月1日・3月15日の2週間ごとの12/20記録です。

6よくある質問

運行管理者試験は難しいですか？
公表合格率30〜40％前後で、科目合格制の足切りがある試験です。例えば5科目中1つでも12/20未満だと不合格になるため、「易しい科目だけ」では通りません。弱点科目を週次で12/20以上に押し上げる計画が有効です。
実務経験があれば簡単ですか？
実務科目は有利になりやすい一方、法令・数値は別途必要です。例えば2026年3月10日CBT想定で実務16/20でも車両法9/20なら不合格です。現場知識を頼りにせず、5科目すべての演習正答数を週1回記録してください。
旅客と貨物で難易度は違いますか？
公表統計では年度ごとに差があります。例えばR7第1回で貨物37.2％・旅客34.1％（公表値）など、自分の区分の数値だけを参考にしてください。合格基準（各60％）自体は区分共通が一般的です（要項で再確認）。
