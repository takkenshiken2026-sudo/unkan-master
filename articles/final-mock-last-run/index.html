運行管理者試験の直前模試｜2週前・100問・維持モード

運行管理者試験の直前模試は、2週前の100問180分計測が最終チェックです。この段階では新規テキストや未読分野の追加は止め、5科目13/20維持と解き直し60％に切り替えます。例えば2026年3月15日CBTなら、3月1日に最終100問模試を入れ、3月8日23時59分変更期限を確認し、3月9日〜14日は解き直し中心で維持モードに入ってください。

この記事の要点

本記事を読むと、直前模試の目的と後半2週の過ごし方が分かり、例えば2026年3月1日模試で5科目13/20以上なら3月9日から新規10問/日までに抑えられます。

1直前模試の目的（維持確認）

直前模試の目的は維持確認です。

直前模試通常模試
13/20維持12/20達成
解き直し60％弱点強化
新規10問/日新規20問/日

例えば2026年3月15日CBTなら、3月1日最終100問模試で5科目13/20以上＋165分以内を確認し、3月9日から維持モードに切り替えてください。この段階で新規教材追加は止めます。

22週前：100問最終模試

2週前：100問最終模試です。

1. 3月1日：100問180分計測
2. 本番同条件（机・時間・通知オフ）
3. 5科目/20＋科目別時間記録
4. 165分以内＋15分見直し
5. 誤答は3日以内解き直し

たとえば2026年3月1日（日）に100問180分最終模試を入れ、「事業14・車両13・道交14・労基13・実務14」と/20記録、3月8日23時59分変更期限をカレンダー登録してください。

3模試後の分析（13/20維持）

模試後の分析（13/20維持）です。

結果対応
5科目13/20以上維持モードへ
1科目12/20その科目+2h/週
1科目11/20以下解き直し優先

例えば2026年3月1日模試で車両法12/20・他4科目13/20以上なら、3月2日〜8日は車両法解き直し20問、他科目は維持10問/日、3月9日から最終1週モードに入ります。CBT日変更は3月8日23時59分までです。

4最終1週：解き直し60％

最終1週：解き直し60％です。

1. 新規テキスト10問/日まで
2. 誤答ノート解き直し60％
3. 数値カード毎日5分
4. 5科目13/20維持
5. 睡眠・体調優先

例えば2026年3月9日〜14日は、1月〜2月の未解き直し誤答と3月1日模試誤答を優先処理し、100問通しは行わず解き直し30問/日を上限にしてください。3月8日23時59分変更期限を確認済みにします。

5CBT前日・当日の確認

CBT前日・当日の確認です。

  • 前日：受験票・身分証・要項確認
  • 前日：会場ルート・到着30分前
  • 当日：6,660円申込済み・CBT予約確認
  • 当日：108秒×100問の時間感覚
  • 当日：5科目12/20以上を意識

たとえば2026年3月15日CBTなら、3月14日に会場確認・身分証・受験票を準備し、3月15日は開始90分前到着、165分解答＋15分見直しのペースを意識してください。要項の持ち物・禁止事項を正本として確認します。

6よくある質問

直前模試は必須ですか？
2週前100問180分計測は強く推奨です。例えば2026年3月15日CBTなら3月1日に最終模試を入れ、5科目13/20以上＋165分以内を確認してください。12/20未満科目が1つでもあれば3月2日〜8日に解き直し優先します。
直前模試で12/20未満が出たら？
新規追加を止め、その科目解き直し30問/週に集中します。例えば2026年3月1日模試で労基法11/20なら、3月2日〜8日に労基法数値カード＋解き直し20問、3月9日から最終1週も労基法10問/日を維持してください。CBT日変更は3月8日23時59分までです。
直前1週は何を優先しますか？
解き直し60％・5科目13/20維持・睡眠です。例えば2026年3月9日〜14日は新規10問/日まで、3月1日模試誤答と1〜2月の未解き直しを優先し、100問通しは行わず数値カード毎日5分を継続してください。
