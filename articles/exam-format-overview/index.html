運行管理者試験の形式｜CBT・5科目・四肢択一

運行管理者試験は令和3年度以降、CBT（Computer Based Testing）で実施されています。パソコン画面の四肢択一で、5科目×各20問程度（要項で再確認）を解き、各科目60％以上（12問以上）を5科目すべて満たすと合格です。例えば残り8週で初めて通し演習を始める場合、5科目100問・180分想定で月2回タイム計測すると本番のペースが掴みやすくなります。正本は運行管理者試験センターの受験要項です。

この記事の要点

本記事を読むと、CBT・5科目・四肢択一の全体像が分かり、例えば1問3分×100問の時間配分を演習に反映できます。

1要項で確認する形式4項目

試験形式の正本は要項です。

項目一般的な案内（要項で再確認）
方式CBT・四肢択一
区分貨物 / 旅客
科目数5科目
問題数各20問程度（計100問前後）
合格基準各科目60％以上
時間90〜180分（要項・年度で確認）

例えば2026年度要項で各科目20問・計100問なら、演習も科目20問単位で正答12問以上を週次記録してください。

2CBTと旧マークシートの違い

令和3年度以前はマークシート試験でした。

  • 現行：PC画面でクリック選択
  • 予約：会場・日時を自分で選択
  • 持ち物：身分証・筆記用具（要項どおり）
  • 禁止：スマートフォン・参考書

たとえば試験2週間前に予約サイトの操作説明動画を15分見ると、当日の「選択し忘れ」「次へ進む」操作で失点するリスクを下げられます。旧来のマークシート練習だけでは不十分です。

35科目と問題数の読み方

5科目は貨物/旅客で事業法科目が異なります。

科目貨物旅客
事業法貨物自動車運送事業法道路運送法
車両法道路運送車両法同左
道交法道路交通法同左
労基法労働基準法同左
実務運行管理実務同左

例えば貨物区分なら事業法8問相当の演習を週1回入れ、正答6問以上（60％）を目標にすると配点の多い分野を押さえやすくなります。

4時間配分の目安

時間配分は要項の試験時間が正本です。

問題数1問あたり目安見直し
100問・180分約1.8分/問最終10分
100問・120分約1.2分/問最終5分
科目20問科目ごと36分（180分÷5）未解答確認

例えば180分想定なら、演習20問を36分以内で解く練習を週2回入れると、本番で労基法6問分の時間不足を防ぎやすくなります。

5形式理解後の演習設計

形式を理解したら演習設計に移ります。

1. 科目別20問→5科目通し100問の順で段階アップ
2. 週次で5科目の正答数（/20）を1行記録
3. 60％未満科目に翌週＋2時間
4. 試験4週前から月2回タイム計測

たとえば残り8週なら、第1〜4週は科目別20問、第5〜8週は100問通しを月2回、という2段階に分けると形式と内容の両方を仕上げられます。試験4週前には5科目すべて12/20以上を1回以上達成しているか確認してください。

6よくある質問

運行管理者試験は何問・何科目ですか？
要項ではCBT・四肢択一で5科目、1科目20問程度が案内されています。例えば各科目20問で60％以上（12問以上）を5科目すべて満たすと合格です。数値は年度・区分で変わるため、申込前に当年度要項の表を開いてください。
マークシート試験はまだありますか？
現行はCBT方式が基本です（令和3年度以降）。例えば2026年度要項でCBTのみと案内されていれば、マークシート記入練習よりPC画面での四肢択一演習を優先してください。方式変更は要項・お知らせで公表されます。
1科目だけ59％でも不合格ですか？
科目合格制のため、1科目でも60％未満なら不合格です。例えば20問中11問正解（55％）の科目があると、他4科目が70％超えでも全体不合格になります。週次で5科目すべて12/20以上を記録する習慣が有効です。
