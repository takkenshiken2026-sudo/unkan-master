運行管理者試験の科目構成｜出題範囲5分野と配点優先順位

科目構成の正本は、要項④「出題分野及び合格基準」の5行表です。現行CBTは法令等5分野・合計30問で、配分は8・4・5・6・7です。構成・範囲・優先順位は本来ひと続きです。この記事では、5行表の写し方から、貨物・旅客の範囲差、配点の優先順位までを1枚で扱います。

1科目構成の正本 — 要項④の5分野30問

出題数は要項④の5行表が正本です。現行CBTは法令等5分野・合計30問で、四肢択一です。

分野出題範囲の軸②最低
（1）8問事業法または道路運送法1問以上
（2）4問道路運送車両法1問以上
（3）5問道路交通法1問以上
（4）6問労働基準法1問以上
（5）7問運行管理の実務知識・能力2問以上

（1）8問と（5）7問の15問で全体の半分を占めます。非公式のまとめサイトより、要項の5行を優先してください。出題数の配分と②最低の列を、同じノート1枚に写すのが構成把握の第一歩です。ただし②最低は出題比とは別の基準です。出題の少ない（2）4問も、毎週1問以上を演習で確認してください。本記事はこの5行を起点に、範囲の中身と配点の優先順位までを順にたどります。

2貨物・旅客の違い — （1）8問の根拠法

（2）～（5）は貨物と旅客で共通です。違うのは（1）8問の根拠法だけで、区分のミスは範囲外の学習を招きます。

区分（1）8問の根拠法テキスト
貨物貨物自動車運送事業法関係貨物版1冊
旅客道路運送法関係旅客版1冊

貨物のCBTを予定する人が旅客用の道路運送法テキストで（1）8問を学ぶと、半分以上が範囲外になります。逆に旅客受験者が貨物自動車運送事業法だけを詰めても、（1）8問は得点になりません。許可証・申込区分・テキストの表紙を、1行のメモで一致させてください。1回の募集で申し込めるのは、貨物か旅客の一方だけです。区分を最初に固定してから、教材選びと演習に入りましょう。

3範囲の論点 — 各法令で問われる中心テーマ

各分野は、法令名称どおりの論点が中心です。詳しくは要項表と教材の目次で照合します。

・（1）8問：許可・届出・運行管理義務・事業者責任
・（2）4問：点検・整備・検査・記録
・（3）5問：運転・装備・速度・違反処理
・（4）6問：労働時間・休息・36協定・安全衛生
・（5）7問：点呼・配車・運行記録・実務判断

（4）6問は数値の暗記が多く、（5）7問は手順や例外を問う形が多い傾向です。誤った問題には分野タグ（1）～（5）を付け、該当章へすぐ戻れる対応表を作りましょう。

4要項5行と教材目次 — 対応づけの4ステップ

試験センターが単一の指定テキストを売っているわけではありません。市販教材は、要項5行と目次を突き合わせて選びます。

1. 要項④の5行をPDFで保存する
2. 区分に合ったテキストの目次を横に並べる
3. 略称や旧称の表記ゆれを1枚に統一する
4. 演習メモの（1）～（5）タグと章名を一致させる

  • 要項の「貨物自動車運送事業法」の行と
  • テキストの事業法編の章を同じ行に書くと
  • （1）8問の誤答から該当章へすぐ戻れ

法改正があった年は注意してください。施行後一定期間は、新旧で解答が変わる問題が原則出題されません。改正反映が安定するまで、要項PDFと公式の改正注記を学習の基準にします。

5配点の優先順位 — ②＞総得点18/30＞出題比

CBTに「科目配点表」はありません。出題比は目安にすぎず、優先順位を決めるのは②の分野別最低正解です。時間が足りない週ほど、次の順序を固定します。

1. ②未達の分野（例：（5）1/7・（2）0/4）
2. 18/30未満の総得点不足
3. 出題比が高い（1）（5）の維持演習
4. 出題比が低い（2）でも②1問は必須

合計で18/30をとっても、（2）0/4のままなら②未達で不合格圏です。②の充足が、総得点18/30より先に効く週もあります。出題比だけで時間を割らず、この3段の順序をぶらさないでください。

6（2）4問の罠 — 出題13%でも最優先級

（2）は出題が4問しかなく、②最低も1問です。だからこそ0/4が②未達に直結し、即不合格圏という罠があります。

・（2）0/4＝②未達で即不合格圏
・（2）1/4＝②はクリアだが得点余地は小さい
・出題が少ないから後回し、は典型ミス
・（1）7/8・合計19/30でも、（2）0/4なら不合格

CBTの30日前になっても（2）が2週連続で0/4なら、車両法の点検章20分＋（2）10問を週3回入れます。まず②1問以上を確保してから、他分野の上積みに移ってください。

7毎週月曜5分 — 優先1分野の入替

優先順位を動かすのは、毎週月曜の5分チェックだけで十分です。

1. 30問シートの分野別正答行を開く
2. 正答が最低の分野や②未達の1分野を選ぶ
3. その分野に+2時間を上書きする
4. 解き直し10問を水曜までに実施する
5. 日曜に30問を通しで解き直す

たとえば前週は（4）弱点に+2h、今週は（2）弱点に+2h、と月曜だけ優先分野を入れ替えると計画がぶれません。安定した（1）8問分は維持演習10問に圧縮してかまいません。CBT4週前は、②未達の分野を集中強化に切り替えます。

8よくある質問

出題数が少ない分野は後回しでよいですか？
後回しにすると②未達で不合格になります。（2）は4問しかなく、0/4なら（1）6/8・合計19/30でも②未達です。出題が少ない分野ほど、1問の取りこぼしが②不合格に直結します。毎週1問以上を演習で確認し、CBT前は②未達の分野を最優先にしてください。出題比よりも②の充足が先です。
貨物テキストで旅客試験は受けられますか？
（1）8問の根拠法が違うため、区分に合ったテキストが必要です。旅客試験は道路運送法関係が中心で、貨物用テキストだけでは（1）8問の対策が足りません。申込区分・テキストの表紙・勤務先の許可を、1行のメモで一致させてから学習を始めてください。1回の募集で申し込めるのは、貨物か旅客の一方だけです。
出題範囲や配点はどこで確認しますか？
運行管理者試験センターの要項④「出題分野及び合格基準」が正本です。unkan.or.jpから当年度PDFを保存し、5行の出題数8/4/5/6/7と②最低を1枚に転記してください。CBTの論点はpast.htmlで確認できます。法改正があった年は、施行後一定期間の出題の扱いを公式の改正注記で確かめてから対策します。
